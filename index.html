Project Idea

Overview

This project would consist of the construction of a synthesiser. The synthesiser would have an analogue interface with digital elements. This project would consist of either building a synthesiser from very basic components, or a more digital and code-centric solution.

Motivation

The motivation for this project comes from an aforementioned interest in music and the desire to build gear. It would certainly be interesting as it encompasses many facets of Information Technology with both connecting hardware through digital and analogue means, as well as programming sounds and functionalities into the device. On top of this, it can be done many different ways. With cheaper Arduino and raspberry pi powered solutions being an easy way to implement this project being one of many ways of executing it; the concept has the flexibility to be more coding dependant than hardware, or to do the opposite.

Description

The synth will need a case fabricated, bare PCB is a bad look as well as dangerous in general for the PCB and the user. The ideal method of input would be for a keyboard style input common among synthesisers with a possibility of drum pads or switches for additional effects; this is what’s known as the controller of the synthesiser. There are additional effects that need to be implemented and modifiable in the final product. They include octave adjustments; envelopes, which controls the attack decay sustain and release of the synthesizer; attack which is the initial part of a note which then lowers down through the decay section; if the note is held it enters into sustain and once it is let go release happens. These components are really important. Another very important component to a synthesiser is a voltage-controlled oscillator, which is what produces the wave, making it the source of the sound for the synthesiser. You pair this with the envelope generator and voltage-controlled filters to help shape the sound. This is all fed into a voltage-controlled amplifier before reaching the onboard amplifier. In short, the oscillator makes the noise, the filter shapes it, the envelope changes how it interacts with the piano keys. The voltage-controlled amp acts as a preamp for the onboard amp. The term voltage control has been mentioned a couple of times and it simply means the sound is adjustable at any point that is voltage controlled which allows for great variability in sound. All the previously listed components allow us to produce the following features:

  1. A set of keys to control the synthesiser.
  2. Different shaped sound waves from an oscillator.
  3. Customisation of Attack, Decay, Sustain and Release.
  4. Ability to move up and down octaves.
  5. A case to house the synth.
  6. Filters to change and shape the oscillators wave.
  7. The amplifiers to make some noise.

All these components are the bare necessities for a simple synthesiser, but there are a few desirable features such as MIDI support, allowing for use as a MIDI controller with a Digital Audio Workstation. Loop functionality would also be a good feature alongside an arpeggiator function.

Tools and Technologies

Depending on the complexity of the approach there are roughly two different potential options. The first option is to use a raspberry pi, Arduino or another board-based technology and equip it with some inputs and outputs, this would mean a mostly digital synthesiser that would also be compact. On the other hand, there is the mostly analogue option, which involves more hardware centric design and involves assembling most of the parts. This appears to not be as feasible or as efficient as the board-based method of creation. The board-based approach would also allow for greater utilisation of the skills relevant to our degree.

Skills Required

For the more likely board-based solution, the team working on the project would need to research and select languages to write and develop the code in, this may be limited by the hardware used such as having to only use select languages with Arduino’s due to compatibility with the Arduino board. In addition, some fabrication and electronics knowledge will be necessary to manufacture housing and install inputs for the synthesiser.

Outcome

The ideal end product is a functioning and usable synthesiser with standard features built with inexpensive resources to a reasonably functional standard and relatively compact form factor, with a clear and repeatable process as well as a large amount of customisation around codable elements.
